**File 1: static/dagre-layout.js (~180-220 lines)**

Create the layout computation module as an IIFE exposing `window.SmartBDagreLayout`:

1. **Text measurement** (internal helpers, not exported):
   - `_measureCanvas` variable (null initially, created lazily)
   - `getMeasureContext()` -- creates canvas, returns 2d context
   - `measureTextWidth(text, fontSize, fontFamily)` -- uses ctx.measureText. Default font: `'600 15px Inter, sans-serif'`
   - `measureNodeDimensions(label, shape)` -- returns `{ width, height }`. Base calculation: textWidth + 32px horizontal padding, 24px line height + 24px vertical padding. Shape-specific adjustments:
     - `circle`: diameter = max(w,h) + 8, set both w and h to diameter
     - `diamond`: multiply both by 1.4
     - `hexagon`: width += height / 2
     - `cylinder`: height += 16 (extra for cylinder cap)
     - All others: no adjustment

2. **Layout computation** (`computeLayout(graphModel)` -- the main function):
   - Create dagre graph: `new dagre.graphlib.Graph({ compound: true })`
   - Set graph options: `{ rankdir: graphModel.direction, nodesep: 60, ranksep: 80, edgesep: 10, marginx: 20, marginy: 20 }`
   - Set default edge label: `function() { return {}; }`
   - **Add nodes**: iterate `Object.entries(graphModel.nodes)`. For each, call `measureNodeDimensions(node.label, node.shape)`, then `g.setNode(nodeId, { label, width, height, shape })`
   - **Add subgraphs**: iterate `Object.entries(graphModel.subgraphs)`. For each subgraph:
     - `g.setNode(sgId, { label: sg.label, clusterLabelPos: 'top', style: 'subgraph' })`
     - For each nodeId in `sg.nodeIds`: `g.setParent(nodeId, sgId)`
     - If `sg.parentId`: `g.setParent(sgId, sg.parentId)`
   - **Add edges**: iterate `graphModel.edges`. For each edge:
     - Check if `edge.from` or `edge.to` is a subgraph key. If so, redirect to the first node in that subgraph's nodeIds (workaround for dagre compound graph bug #238). Store the original endpoint for later.
     - Set edge: `g.setEdge(from, to, { label: edge.label || '', width: edge.label ? measureTextWidth(edge.label) + 16 : 0, height: edge.label ? 20 : 0, labelpos: 'c' })`
   - **Run layout**: `dagre.layout(g)`
   - **Extract results**: call `extractLayoutResult(g, graphModel)`

3. **Result extraction** (`extractLayoutResult(g, graphModel)`):
   - Returns a `LayoutResult` object with:
     - `width`: `g.graph().width`
     - `height`: `g.graph().height`
     - `nodes`: array of `{ id, label, shape, x, y, width, height }` from `g.nodes()` (skip subgraph nodes)
     - `edges`: array of `{ id, from, to, label, type, points }` from `g.edges()` where `points` comes from `g.edge(e).points`
     - `subgraphs`: array of `{ id, label, x, y, width, height }` from subgraph nodes in `g.nodes()`

4. **Public API**: `window.SmartBDagreLayout = { computeLayout: computeLayout }`

**File 2: static/viewport-transform.js (~80-120 lines)**

Create a ViewportTransform class as an IIFE exposing `window.ViewportTransform`:

1. **Constructor**: `function ViewportTransform()` -- initializes `this.x = 0`, `this.y = 0`, `this.zoom = 1`

2. **screenToGraph(screenX, screenY)**: returns `{ x: (screenX - this.x) / this.zoom, y: (screenY - this.y) / this.zoom }`

3. **graphToScreen(graphX, graphY)**: returns `{ x: graphX * this.zoom + this.x, y: graphY * this.zoom + this.y }`

4. **zoomToFit(graphWidth, graphHeight, containerWidth, containerHeight)**:
   - `padFraction = 0.92`
   - `scaleX = (containerWidth * padFraction) / graphWidth`
   - `scaleY = (containerHeight * padFraction) / graphHeight`
   - `this.zoom = Math.min(scaleX, scaleY, 2.5)`
   - Center: `this.x = (containerWidth - graphWidth * this.zoom) / 2`, same for y

5. **applyToElement(element)**: sets `element.style.transform = 'translate(' + this.x + 'px, ' + this.y + 'px) scale(' + this.zoom + ')'`

6. **setTransform(x, y, zoom)**: sets all three values directly

7. **getTransform()**: returns `{ x: this.x, y: this.y, zoom: this.zoom }`

8. **Public API**: `window.ViewportTransform = ViewportTransform`

Use `var` throughout (matching codebase convention for static/*.js). Use `'use strict'` in the IIFE.
